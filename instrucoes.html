INSTRUÇÕES JOGO DESENVOLVEDORES

COMO JOGAR:

Para este jogo, você assumirá o controle dos dois cowboys, cada um será controlado individualmente.

Para o cowboy 1: use as teclas W e S para se mover e a tecla Space para atirar.

Para o cowboy 2: use as setas para cima e para baixo para se mover e a tecla Enter para atirar.

OBJETIVO DO JOGO:

No velho Oeste, dois cowboys competem para ser os melhores pistoleiros. Uma guerra foi declarada.

Nesse jogo, você assumirá o controle dos cowboys e deverá decidir o final dessa briga em uma batalha.

Jogue com os cowboys!
